## 基于Java技术的物流公司管理系统设计与实现

它能够更好帮助物流企业完成进、销、存一体化信息管理，并配有完善的配送、运输追踪、报价等功能。 

### 设计知识点：

1. 了解相关的一些调研分析方法
2. HTML、CSS、JavaScript编程；
3. Java编程技术；
4. Mysql数据库的使用；
5. SSM或Spring Boot框架开发；
6. 软件工程设计方法；
7. 数据库设计方法
8. 掌握软件测试的方法；

### 主要设计内容：

1. 用户可以自助下单，系统根据重量和目的地以及发货地提供报价
2. 管理员可以更改各个发货地到目的地的价格
3. 管理员可以修改重量与价格关系
4. 管理员可以添加删除和修改城市名称
5. 各中转点可以对物流的状态信息更改，并更新物流状态
